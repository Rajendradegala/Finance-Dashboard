# Finance Dashboard UI

Overview:

This project is a simple finance dashboard built using React. It allows users to track their financial transactions, view summaries, and understand spending patterns.

Features:

* Dashboard summary (Balance, Income, Expense)
* Transactions list
* Add new transactions (Admin role)
* Role-based UI (Viewer/Admin)
* Filtering transactions (Income/Expense)
* Insights (highest spending category)

Tech Stack:

* React (useState)
* CSS for styling

How to Run:

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000

Approach:

I focused on building a clean and simple UI with core functionality. The application uses React state to manage transactions and dynamically update the UI.

Future Improvements:

* Add charts for better visualization
* Add local storage for data persistence
* Improve UI with animations
